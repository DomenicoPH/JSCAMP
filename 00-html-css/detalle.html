Ingeniero de Software

Tech Solutions Inc. | Remoto

Descripción del puesto

Tech Solutions Inc. está buscando un ingeniero de software altamente motivado y experimentado para unirse a nuestro equipo remoto. El candidato ideal tendrá una sólida formación en desarrollo de software, con experiencia en diseño, desarrolo e implementación de soluciones de software escalables y de alto rendimiento. Como ingeniero de software usted será responsable de liderar proyectos de desarrollo, mentorizar a ingenieros junior y colaborar con equipos multifuncionales para entregar productos de software de alta calidad.

Responsabilidades

  • Diseñar, desarrollar y mantener aplicaciones web utilizando tecnologías modernas.
  • Colaborar con equipos de productos y diseño para definir y entregar nuevas características.
  • Escribir código limpio, eficiente y bien documentado.
  • Realizar revisiones de código y proporcionar retroalimentación constructiva a los miembros del equipo.

Requisitos

  • Licenciatura en informática o campo relacionado.
  • Mínimo de 5 años de experiencia en desarrollo de software.
  • Experiencia con frameworks de JavaScript (React, Angular, Vue.js).
  • Familiaridad con metodoloías ágiles y herramientas de control de versiones (Git).

Acerca de la empresa

Tech Solutions Inc. es una empresa de tecnología innovadora que se centra en la creación de soluciones de software de vanguardia para diversas industrias. Estamos comprometidos con el fomento de un entorno de trabajo colaborativo e inclusivo donde cada empleado pueda prosperar y crecer profesionalmente.

Ofrecemos salarios competitivos, beneficios integrales y oportunidades de desarrollo profesional continuo.
